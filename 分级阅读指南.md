## 🎯 学习建议

### 给 Rust 新手的特别提示

1. **不要纠结于所有语法细节**
   - 第一遍理解大致意思即可
   - 遇到不懂的类型，先看注释

2. **理解所有权是关键**
   - `T`: 拥有所有权
   - `&T`: 不可变借用
   - `&mut T`: 可变借用
   - 函数参数常用借用

3. **Result 和？是朋友**
   - `Result<T, E>`: 可能失败的操作
   - `?`: 失败时自动返回
   - 比 try-catch 更简洁

4. **多查看 Cargo.toml**
   - 了解使用了哪些库
   - 理解版本选择

5. **利用编译器错误信息**
   - Rust 编译器错误很友好
   - 通常会告诉你如何修复

### 推荐学习工具

1. **Rust Playground**: https://play.rust-lang.org/
   - 在线运行 Rust 代码
   - 测试小片段

2. **Rust By Example**: https://doc.rust-lang.org/rust-by-example/
   - 通过例子学习 Rust
   - 有中文版

3. **Rust Reference**: https://doc.rust-lang.org/reference/
   - 官方参考手册
   - 查询语法细节
