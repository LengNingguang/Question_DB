# 题目分数功能实现工作报告

## 一、任务概述

**需求**：为 CPHOS Question_DB 题库系统实现"题目分数"功能
- 数据库添加题目分数字段
- POST/PUT 上传时从 TeX 文件解析分数
- 搜索时支持分数范围过滤（score_min/score_max）
- 同步更新集成测试和 API 文档

**分数规范**：
- 类型：`INTEGER`（整数）
- 范围：0-400（包含）
- 可选：允许 `NULL`
- 来源：从 TeX 文件 `\begin{problem}[分数]{title}` 解析

## 六、设计决策说明

| 决策 | 理由 |
|------|------|
| 分数存储为关系型列（非 JSON） | 便于 SQL 过滤查询，符合项目现有模式 |
| PATCH 不支持修改 score | 分数是 TeX 文件属性，应通过替换文件更新 |
| PUT 自动更新 score | 替换 TeX 后重新解析，保持数据一致 |
| 分数可选（允许 NULL） | 兼容旧题目，解析失败不阻断上传 |
| 正则匹配两种 TeX 格式 | 支持 `\begin{problem}[score]` 和 `\problem[score]` |
